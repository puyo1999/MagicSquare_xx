# MagicSquare_1004 — 10축 검증 체크리스트 (Skill)

| 항목 | 내용 |
|------|------|
| 대상 | 4×4 부분 마방진 학습자 (Mom Test 2차) |
| 목적 | “행 합 34만” 착각 방지 — **삽입 후** 10축 판정 |
| Command | `validate_lines(grid)` — R1~R4 · C1~C4 · D1 · D2 |
| SSOT | `docs/PRD.md` FR-15 · `Report/03.*` |

---

## 사용 시점

- [ ] 빈칸을 모두 채운 뒤 (**16칸 완성**)
- [ ] 종이에서 행 합 34만 확인한 직후 (**아직 완료 아님**)
- [ ] 5번 바꿔넣기 후 제출 직전

---

## 절차

| # | 확인 | 체크 |
|---|------|------|
| 1 | 격자에 `0`(빈칸)이 **없는가**? | ☐ |
| 2 | 빈칸이 있으면 → `incomplete` (10축 판정 **하지 않음**) | ☐ |
| 3 | `validate_lines(grid)` 호출 | ☐ |
| 4 | `status == "pass"` 인가? | ☐ |
| 5 | `fail`이면 `failed_lines`에서 깨진 축 확인 (R / C / D) | ☐ |
| 6 | **행만** 맞아도 pass가 **아님** — 열·D1·D2 반드시 포함 | ☐ |
| 7 | 중복·17 등 범위 위반은 **R2 / D-003** — 이 체크리스트와 별도 | ☐ |

---

## 10축 ID 빠른 참조

| 축 | ID | 합 |
|----|-----|-----|
| 행 1~4 | R1, R2, R3, R4 | 34 |
| 열 1~4 | C1, C2, C3, C4 | 34 |
| 주대각 | D1 | 34 |
| 반대각 | D2 | 34 |

---

## Mom Test 연결 (한 줄)

> 행 합 34 우선 후 “넣어본 뒤” 열·대각을 확인하지 않으면 SC-1 갭이 남는다 — 이 체크리스트는 **코드 자동화가 아니라 행동 규칙**이다.

---

## 자가 점검

| 질문 | 예 → 다음 |
|------|-----------|
| 행 4개만 34인가? | **아직 끝 아님** → `validate_lines` |
| `failed_lines`에 `C*`만 있는가? | 열 수정 후 재검증 |
| `D1` 또는 `D2`만 있는가? | 대각 수정 후 재검증 |
| `status == "pass"`? | 제출 후보 OK (R2 별도 확인) |
