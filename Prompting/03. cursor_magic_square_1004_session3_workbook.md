## 1) 주제 한 문장

> 4×4 부분 마방진을 채운 뒤, “행만 맞으면 된다”고 느끼는 순간과 실제로 열·대각선이 깨지는 순간을 구분할 수 있게, 채워진 격자가 마방진 조건을 만족하는지 판정하는 기준을 먼저 정한다.

## 2) R-G-I-O

| | 내용 |
|---|------|
| **Role** | 1월 과제를 손으로만 풀었던 학습자 — 행 합 34 우선, 삽입 후 열·대각 확인, 맞을 때까지 바꿔넣기(약 5회) |
| **Goal** | 채워진 4×4 격자에 대해 행·열·대각선 합 34와 1~16 중복 없음을 한 번에 판정할 수 있는 검증 규칙·명령·테스트 루프를 Session 3 범위에서 고정 |
| **Input** | 4×4 정수 격자(1~16, 빈칸 없음). Mom Test 재현: 행은 34인데 열/대각선이 깨진 1월 “반대로 넣었더니” 패턴 |
| **Output** | pass/fail + 어느 축(행/열/대각/중복)이 깨졌는지 — Rule · Command · Test Loop (구현 코드는 다음 세션) |
