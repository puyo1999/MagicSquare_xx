## Mom Test 워크북 — MagicSquare_xx (STEP 1 · 이번 인터뷰)

### 페르소나:
4×4 부분 마방진(빈칸 2개, 합 34)을 **손으로** 푸는 학습자. **코드로 다룬 적 없음**. 어려운 것부터 처리하는 습관 있음.

### 인터뷰 답변 (과거 사실):
- 올해 **1월**, 슬라이드와 같은 문제 **제출 완료**, **약 25분**
- 후보군 찾기 **약 24분**(종이, **행 합 34 우선**), 넣어보기 **약 30초**
- 열·대각선은 **넣어 본 뒤** 확인; **5번** 바꿔넣기(**맞을 때까지**)
- **손으로만**, 1월 이후 같은 문제 **없음**, **코드 없음**
- 후보 **1개**만 찾음(빈칸당·조합 **미확인**)

### 표면 문제 (금지 — 솔루션 섞인 문장):
- “4×4 마방진 **검증 프로그램**을 만든다”
- “ECB로 **SquareValidator**를 만든다”
- “대각선을 **자동으로 체크하는 앱**을 만든다”
- “**Solver**로 빈칸 2개를 **자동 채운다**”

### 진짜 문제 (한 문장):
손으로 4×4 부분 마방진을 풀 때 **행 합 34만**으로 빈칸 후보를 종이에 적어 좁히는 데 **약 24분**을 쓰고, **열·대각선은 넣어 본 뒤에야** 확인하며 **왜 틀렸는지 보기보다** 빈칸 숫자를 **맞을 때까지 바꿔 넣는** 방식으로 **약 25분**을 쓴다.

**Mom Test 증거 3줄**
1. “**행 합 34 우선**” / “**넣어본 뒤에**”
2. “**맞을 때까지 바꿔본 것**” / “**5번**”
3. “**후보군 찾기**” / “**적어내며**” / “후보 **1개밖에**”
