## Mom Test 결과 (입력)

### 페르소나

4×4 부분 마방진(빈칸 2개, 합 34)을 **손으로** 푸는 학습자. **코드로 다룬 적 없음**. 어려운 것(대각선 등)부터 처리하는 습관 있음.

### 진짜 문제 (한 문장)

손으로 4×4 부분 마방진을 풀 때 **행 합 34만**으로 빈칸 후보를 종이에 적어 좁히는 데 **약 24분**을 쓰고, **열·대각선은 넣어 본 뒤에야** 확인하며 **왜 틀렸는지 보기보다** 빈칸 숫자를 **맞을 때까지 바꿔 넣는** 방식으로 **약 25분**을 쓴다.

### Mom Test 증거 3줄

1. “**행 합 34 우선** 기준을 잡았어” / “**넣어본 뒤에**” (열·대각선 확인)
2. “**맞을 때까지 바꿔본 것**이지, 안 되는 것을 먼저 찾기보다는” / “**5번** 정도 시도”
3. “**후보군 찾기**에 시간을 보냈지” / “**적어내며** 했었지” / “후보는 **1개밖에 못 찾았어**”

## 1) 주제 한 문장 (Mom Test 기반, 솔루션 최소화)

> **4×4 부분 마방진을 채운 뒤, “행만 맞으면 된다”고 느끼는 순간과 실제로 열·대각선이 깨지는 순간을 구분할 수 있게, 채워진 격자가 마방진 조건을 만족하는지 판정하는 기준을 먼저 정한다.**

## 2) R-G-I-O (Role / Goal / Input / Output)

| | 내용 |
|---|------|
| **Role** | 1월 과제를 **손으로만** 풀었던 학습자 — 행 합 34 우선, 삽입 후 열·대각 확인, 맞을 때까지 바꿔넣기(약 5회) |
| **Goal** | 채워진 4×4 격자에 대해 **행·열·대각선 합 34**와 **1~16 중복 없음**을 한 번에 판정할 수 있는 **검증 규칙·명령·테스트 루프**를 Session 3 범위에서 고정한다 |
| **Input** | 4×4 정수 격자(1~16, 빈칸 없음). Mom Test 재현 케이스: **행은 34인데 열/대각선이 깨진** 1월 “반대로 넣었더니” 패턴 |
| **Output** | **pass / fail** + **어느 축(행/열/대각/중복)이 깨졌는지**를 말할 수 있는 Rule · Command · Test Loop (구현 코드는 다음 세션) |

## 5) 8계층 중 이번 세션에서 만드는 것

| 계층 | 이번 Session 3 산출 | Mom Test / Validator 연결 |
|------|---------------------|---------------------------|
| **Rule** | 4×4, 1~16 각 1회, **10개 축**(4행 + 4열 + 2대각) 합 **34** → pass. 하나라도 불일치 또는 중복/범위 위반 → fail + **위반 축 명시** | “넣어본 뒤에” 확인했던 **열·대각**을 Rule에 **판정 시 필수**로 고정 |
| **Command** | `validate(grid)` — 입력: 4×4 채워진 격자 → 출력: `{ pass \| fail, violations[] }`. Skill에서 호출하는 **단일 진입점**만 정의 | 5번 바꿔넣기 **후** 제출 직전 “대각선부터” 검산을 Command **호출 순서**로 모델링 가능 |
| **(Skill)** | *선택* Cursor Skill 또는 체크리스트: “행 34만 보면 안 됨 → Command로 **전 축** 검증” | 증거 ① 행 우선 습관을 **Skill로 상쇄** (자동화 X, 행동 규칙만) |
| **Test Loop** | **D-001** 완전 마방진 → pass · **D-002** 행=34, 열/대각 깨짐(1월 ‘반대로 넣음’) → fail, violations · **D-003** 중복 또는 1~16 밖 → fail. Red → Rule/Command 수정 → 재실행 | SC-1~3 및 Mom Test 3줄을 **재현 가능한 시나리오**로 고정 |

### Test Loop 시나리오 (D-001~D-003)

| ID | 시나리오 | 기대 |
|----|----------|------|
| **D-001** | 표준 4×4 완전 마방진 (1~16, 빈칸 없음) | **pass**, violations 없음 |
| **D-002** | **4행 모두 합 34**이나 **한 열 또는 대각선 ≠ 34** (1월 “반대로 넣었더니” 재현) | **fail**, `violations`에 열 또는 대각선 |
| **D-003** | 숫자 **중복** 또는 **1~16 밖** 값 | **fail**, `violations`에 duplicate / range |
